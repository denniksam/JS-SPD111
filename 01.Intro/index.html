Вступ до JavaScript

JavaScript - мова програмування, відпочатку створена для HTML, але на сьогодні - самостійна мова. JS - вільна (від ліцензій) мова, але сама назва ліцензована. Для публікації стандартів мови вживається термін ECMA-Script (ES).

Кажучі про JS мається на увазі дві реалізації мови: браузерний варіант та NodeJS - збірка самостійної мови. Це мова-інтерпретатор або REPL (Read-Eval-Print Loop).

Для того щоб запустити JS у браузері є декілька можливостей.

  • Включивши код в атрибути полів HTML-елементів, які відповідають за оброблення подій, наприклад, атрибут onclick
  • Додавши секцію <script></script> з кодом у неї
  • Створивши окремий файл-скрипт та підключивши його тим же тегом <script src='...'></script> (дозволяється лише одне - або тіло, або файл тегу)
  • Написавши інструкцію у консолі браузера

Змінні та типи даних. JS є прототипною мовою - це різновид ООП, але з рядом особливостей. Усе є об'єкт, але побудований він не з типу, а з прототипу. Тобто береться певна "основа" і до неї додаються довільні елементи. Змінні з'являються у момент запису і не вимагають окремого оголошення. Змінні можуть змінювати тип даних, що у них зберігається, при наступних присвоюваннях.

Традиційний спосіб запуску скриптів - описувати їх в окремому файлі, підключати його, а в обробниках подій звертатись до функцій, описаних у скриптах.
